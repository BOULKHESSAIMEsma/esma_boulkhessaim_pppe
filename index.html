Screen Time vs Mental Wellness Predictor

Developed a machine learning web application that predicts mental wellness index based on screen time habits and lifestyle factors. The model analyzes 13 different features including age, screen time hours, sleep quality, stress levels, and exercise habits to provide personalized mental wellness predictions. Built using Linear Regression with scikit-learn and deployed as an interactive Flask web application where users can input their data and receive real-time predictions about their mental wellness score.

Python Flask scikit-learn Pandas Linear Regression Matplotlib

Diabetes Prediction System

Built a supervised learning classification system to predict diabetes risk using patient health metrics. Implemented and compared Decision Tree and Random Forest algorithms with extensive hyperparameter tuning including max_depth, min_samples_split, and class_weight balancing. The model analyzes 8 medical features such as glucose levels, BMI, blood pressure, and age to classify diabetes risk. Deployed as a user-friendly Flask web application with color-coded prediction results and an intuitive interface for healthcare professionals.

Python Flask scikit-learn Decision Tree Random Forest Classification Healthcare ML
